# Personal Portfolio Website

This is a personal portfolio website built for **RIO**, an Informatics Management student specializing in Android and Web Development. The website is designed to showcase projects, skills, articles, and client collaborations.

## 🚀 Features

- **Responsive Design**: Fully responsive layout that works flawlessly on desktop, tablet, and mobile devices.
- **Dark/Light Mode**: Built-in dark mode support that respects system preferences and allows manual toggling.
- **Dynamic Typing Effect**: Uses `Typed.js` for an engaging typing animation in the hero section.
- **Project Showcase**: Displays highlighted projects (e.g., MiDClass App, SmartDrum).
- **Blog Section**: Links to recent articles on technology and tutorials.
- **Contact Form**: Functional contact form integrated with Formspree.

## 🛠️ Technologies Used

- **HTML5**: Semantic markup for the structure of the website.
- **Tailwind CSS**: Utility-first CSS framework for rapid UI development and styling.
- **JavaScript (Vanilla)**: For interactive elements like the hamburger menu, dark mode toggle, and "Back to Top" functionality.
- **Typed.js**: A JavaScript library for the typing animation effect.

## 📂 Project Structure

- `index.html`: The main entry point of the website.
- `package.json`: Contains project dependencies (Tailwind CSS, Prettier) and metadata.
- `dist/`: Directory containing compiled CSS (`dist/css/final.css`), JavaScript (`dist/js/script.js`), and static assets like images and SVG icons.

## ⚙️ Getting Started

### Prerequisites

To modify the styles or contribute to this project locally, make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone this repository or download the source code.
2. Open the project directory in your terminal:
   ```bash
   cd Portofolio
   ```
3. Install the required development dependencies (Tailwind CSS, Prettier):
   ```bash
   npm install
   ```

### Running Locally

Since this is a static website, you can simply open the `index.html` file in your preferred web browser to view it. 

For a better development experience, it is recommended to use an extension like **Live Server** in Visual Studio Code.

### Compiling Tailwind CSS

If you make changes to the HTML classes or the Tailwind configuration, you need to recompile the CSS. Ensure you have the proper Tailwind CLI command set up to output to `dist/css/final.css`.
